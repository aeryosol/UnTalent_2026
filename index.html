Latihan Uji Kompetensi Teknis

Platform latihan soal berbasis AI untuk persiapan uji kompetensi jabatan struktural Kantor Pelayanan Pajak — Direktorat Jenderal Pajak.

60 Soal per Sesi 40 Pilihan Ganda 4 Kasus × 5 Soal Pembahasan AI
1
Buka Anthropic Console Gratis
Kunjungi console.anthropic.com lalu klik Sign up. Daftar dengan email atau akun Google.
2
Verifikasi nomor HP
Anthropic mengirim SMS kode verifikasi. Masukkan kodenya untuk mengaktifkan akun.
3
Buka menu API Keys
Setelah login, klik API Keys di sidebar kiri — atau langsung ke console.anthropic.com/settings/keys.
4
Buat API key baru
Klik Create Key, beri nama bebas (misal: djp-quiz), lalu klik Create Key.
5
Salin key-nya sekarang
Key hanya ditampilkan sekali. Salin teks yang dimulai dengan sk-ant-... dan tempelkan ke kolom di bawah.
Info biaya: Akun baru mendapat free credits yang cukup untuk ratusan sesi latihan. Tagihan hanya muncul jika credit habis dan kamu menambahkan kartu kredit. Pantau penggunaan di console.anthropic.com/settings/cost.

Masukkan Anthropic API Key

Soal dibuat secara dinamis oleh AI. API key hanya tersimpan di memori browser kamu, tidak dikirim ke mana pun selain Anthropic.

Belum punya API key? Ikuti panduan di atas untuk mendapatkannya secara gratis.
